lemongrab
=========

Simple XMPP bot, written in C++, made for private use

[![Build Status](https://travis-ci.org/Chemrat/lemongrab.svg?branch=master)](https://travis-ci.org/Chemrat/lemongrab)

List of modules:

* URL preview (YouTube, Steam)
* Dice roller
* Last seen

Dependencies:

* gloox
* curl
* leveldb
